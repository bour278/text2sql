
1. Navigate to the project root.
   
2. Create the file:
   ```bash
   touch sandbox/keys.env
   ```
   
3. Open `sandbox/keys.env`.

4. Add the line:
   ```
   OPENAI_API_KEY=YOUR_OPENAI_API_KEY
   ```
5. Save the file.

6. Navigate to the `src` directory:
   ```bash
   cd sandbox
   ```

7. Run the script:
   ```bash
   python test-langchain.py
   ```
